# Introduction 
The product is intended to provide an entry-level Accounts Payable. To achieve this we will leverage our knowledge and experience in account payable automation to support the development of the CBS AP product.

# Getting Started

# 1. Required Tools
> * `Install Node JS with LTS package recommended by user` -    https://nodejs.org/en/download
> * `Install angular cli` -  https://angular.io/cli
 **Note:** The angular version of this application is v16
  
# 2.  Clone the Repository
```shell
git clone https://CBS-Australia@dev.azure.com/CBS-Australia/CBS%20AAP/_git/cbs-ap-web
cd cbsaap-web
```
# 3. Install npm package
```shell
npm install
npm start
```
 # 4. Check for the latest release of angular
  [Latest update v16](https://angular.io/guide/update-to-version-16)
 # 5. For angular security guildelines
